# ikarthiksekar.github.io

Personal resume website for **Karthik Sekar** — Quality Engineering Lead, AI & Automation — hosted via GitHub Pages.

## Live Site

https://ikarthiksekar.github.io

## Contents

- `index.html` — Resume page markup
- `style.css` — Styling for the resume layout
- `README.md` — This file

## Local Preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deployment

Changes pushed to the `main` branch are automatically published by GitHub Pages.
